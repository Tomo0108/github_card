GitHub Commands

> git add . へんこうをステージについかする
> git commit -m "" へんこうをきろくする
> git push リモートにアップロードする
> git init リポジトリをしょきかする
> git clone リモートリポジトリをコピーする
> git pull リモートのへんこうをしゅとくする
